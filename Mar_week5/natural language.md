# 모아쓰기와 풀어쓰기

## 한국어 글자 수준 토큰화 
- 한글에는 모아쓰기(예:밝)와 풀어쓰기(예:ㅂㅏㄺ)가 있음
- 현대 한국어 모아쓰기 글자 수 : 11,172
- 현대 한국어 풀어쓰기 글자 수 :
    - 초성 19글자
    - 중성 21글자
    - 종성 28글자(종성없음도 1글자로 포함할 경우.)
- 한국어 모아쓰기 <-> 풀어쓰기 간 변환 : 유니코드 정규화를 이용하여 간단히 할 수 있음.
